Página web dedicada a mostrar mis proyectos de la asignatura "Desarollo Web Entorno Cliente"

- Primer programa: La finalidad de este programa es comprobar el tipo de variable. El usuario elige una de las opciones, se envia 
el valor seleccionado al programa y éste muestra por pantalla que tipo de variable ha elegido el usuario.

- Segundo programa: La finalidad de este programa es mostrar el alcance de las variables let, var y const sobre un programa, para conseguir eso
en cada uno de los casos se da un valor inicial a la variable, se introduce un nuevo valor a la variable en una estructura interna diferente y
se observa si al salir de esta estructura interna el valor inicial de la variable varia o se mantiene igual.

- Tercer programa: La finalidad de este programa es simular un formulario de registro y de inicio de sesión. Si dejas campos vacios te muestra
mensajes de error, si al registrarte la contraseña no coincide con la repetición de la contraseña salta un error, si todo esta correcto se guarda
el usuario en un array, pero si el usuario y contraseña ya están registrados salta un mensaje de error. Y si no estas registrado no puedes iniciar sesión, 
te salta un mensaje de error pero si estas registrado te da la bienvenida.
